## Important rules

- Do not remove local compatibility work for Claude Code, Codex, Cherry Studio, Kimi, GLM, GPT images, or OpenAI Responses without explicit confirmation.
- Do not reintroduce the upstream Affiliate / invite-rebate module unless the maintainer explicitly requests it.
- Do not overwrite the custom Promotion center, usage-log enhancements, billing multipliers, model mapping, auto-ops, proxy-pool, or backup-lock behavior.
- Do not commit real credentials, server IPs, database dumps, Redis data, Clash/proxy configs, S3 keys, or `.env` files.

This file intentionally replaces the upstream CLA text because this fork is maintained independently and should not imply that custom contributions are being submitted directly to the upstream project owner.
